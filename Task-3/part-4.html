Block element

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

  • Two commonly used block elements are: p and Div.
  • The p element defines a paragraph in an HTML document.
  • The div element defines a division or a section in an HTML document.

Inline element

An inline element does not start on a new line. An inline element only takes up as much width as necessary. This is a span element inside a paragraph.

  1. The inline-level span only occupies as much width as necessary, and its margin and padding may not affect the layout as significantly as with block-level elements.
